# 🗳️ Election Management System

An interactive **full-stack web application** that simulates the complete election process — from voter registration to real-time result analysis.

Built as part of **PromptWars Challenge 2 – Election Process Education (#BuildwithAI)**.

---

## 🚀 Live Demo
🔗 https://election-website-1044066290418.us-central1.run.app  

---

## 📌 About the Project

Understanding elections can be confusing, especially for first-time voters.  
This project provides a **hands-on simulation** of how elections actually work — allowing users to experience the entire process instead of just reading about it.

Inspired by the idea that **interactive systems improve civic awareness and engagement** 0

---

## ✨ Features

- 🔐 **User Authentication** – Secure registration & login  
- 👤 **Candidate Management** – View candidate profiles and details  
- 🗳️ **Voting System** – Cast votes through an interactive interface  
- 📊 **Live Results Dashboard** – Real-time vote counting & winner display  
- 🧠 **Election Quiz** – Test and improve understanding of election concepts  
- 📱 **Responsive Design** – Works across mobile and desktop devices

## ⚙️ Installation & Setup

1. Clone the repository
```bash
git clone https://github.com/Aditya9236/Election-Education.git
```

2. Navigate into the project folder
```bash
cd Election-Education
```

3. Install dependencies
```bash
npm install
```

4. Start the server
```bash
npm start
```

5. Open in browser
```
http://localhost:3000
```
